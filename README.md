# 相亲模拟器：面子与里子 (The Blind Date: Face & Fate)

一款 Roguelike 文字模拟游戏，探讨现代相亲文化中"自我价值"与"社会期待"之间的错位。

## 🎮 游戏概述

**类型**: Roguelike 文字模拟 / 策略养成 / 心理博弈

**核心体验**: 在随机的"天崩"或"天胡"开局下，通过资源管理（真金白银 vs. 虚假包装）和话术博弈，在相亲桌上生存下来。

**Slogan**: "真诚是必杀技，但如果你没有，那就靠演技。"

## 🚀 快速开始

### 安装依赖

```bash
cd blind-date-simulator
npm install
```

### 开发模式

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

## � 核心玩法

### 角色系统
- **财富值**: 决定初始资金
- **颜值**: 影响初始印象分
- **家庭背景**: 相亲中的"硬通货"
- **工作履历**: 影响每回合资金回复
- **心理素质**: 决定撒谎时的穿帮率

### 角色原型
- 🎲 **完全随机**: 命运的骰子
- 💰 **富二代**: Easy 模式
- 🔥 **凤凰男**: 寒门逆袭
- 👤 **普通人**: 平平淡淡
- 💀 **天崩开局**: Hard 模式

### 道具系统
- **真实资产**: 高价，无穿帮风险
- **虚假包装**: 低价，有穿帮概率
- **特殊道具**: 有特殊效果或副作用

### 对话战斗
- 回合制对话
- 四种回应策略：诚实 / 撒谎 / 转移话题 / 使用道具
- 穿帮判定机制

### 女嘉宾类型
- 💵 **务实派**: 看重经济条件
- ✨ **颜控派**: 颜值即正义
- 💕 **灵魂伴侣派**: 看重三观契合
- 👑 **顶级猎手**: BOSS 级难度

### 多结局系统
- 结局 A: 虚假的繁荣
- 结局 B: 老实人的悲歌
- 结局 C: 意外的真爱 (隐藏结局)
- 结局 D: 社死现场

## 🛠️ 技术栈

- **框架**: React 18 + TypeScript
- **状态管理**: Redux Toolkit
- **构建工具**: Vite
- **样式**: CSS3 + CSS Variables

## 📁 项目结构

```
blind-date-simulator/
├── src/
│   ├── components/     # React 组件
│   ├── store/          # Redux 状态管理
│   ├── systems/        # 游戏系统逻辑
│   ├── types/          # TypeScript 类型定义
│   └── utils/          # 工具函数
├── data/
│   ├── items/          # 道具配置
│   ├── npcs/           # NPC 配置
│   ├── events/         # 事件配置
│   └── endings/        # 结局配置
└── public/             # 静态资源
```
